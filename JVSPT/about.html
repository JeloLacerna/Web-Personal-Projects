About

The Project

This project is first and foremost, a requirement for course I am doing for my Web and Mobile App Development Program.

As afan of Hoyoverse's Honkai Impact 3 and Genshin Impact games, I decided to create this project.

The main goal of this page is to be a centralized place to get information about the game and its mechanics.

This project may not have ALL the information there is, but the information present is concise and essential.

If you have any questions or suggestions, use the contact form or send me a mail: lacerna.jelouie.24@gmail.com
