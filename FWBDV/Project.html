Tower of Hanoi Game with Autosolver Programming Language: Java

This project is a Tower of Hanoi GUI game with account validation, save and load game options, and an animated solution to the current game.

This program uses JavaFX API to draw the UI to the user. It also employs JDBC connected to a MySQL database for account validation. The animation is a series of Translation transition inside a Sequential transition. The menu UI is built using Scene builder and FXML sheets, and the game UI is built manually using JavaFX statements.

Vehicle Life Simulator Programming Language: Java

This project is a vehicle simulator similar to a life simulator. The simulator is comprised of a field with 4 quadrants and vehicles of 4 types - car, truck, sports car, and tractor. These vehicles will follow their own movement every turn based on which type they are. The field is populated by a number of these vehicles in random fashion.

The vehicles move on a set pattern based on their type: cars move by 1 unit per turn, trucks by 2 units on either a single direction or on both directions, sports cars by 1 unit but given 3 cycles each turn, and tractors by 1 unit but given 2 cycles each turn. If the vehicles go over the edge of the field, they are ‘killed’. A ‘dead’ vehicle will no longer move on the next turn. The simulation will end if all vehicles are ‘dead’ or if the number of turns is exhausted.

The simulator is what governs all processes with inside the simulation itself. This class ‘runs’ the simulation and records all the path each vehicle object took during the simulation. Each turn the new status of the vehicle objects are printed. After going through all of the simulation cycles, a summary for every vehicle object is printed complete with the number of turns alive and the coordinates they took during the simulation. Finally, the top 3 vehicle objects of each type are shown to the user.

ATM Simulator Programming Language: C#

This project is an Automatic Teller Machine simulator. This simulator is able to offer various transactions to the user such as withdrawal, deposit, bills payment and funds transfer. The ATM can also be serviced by an Administrator.

This program uses UI elements created using windows forms complete with a screen, number pads and buttons.
